# Study Planner Algorithm

## Purpose

Generate a personalized SAT study schedule based on:

- Current score
- Target score
- Test date
- Available study time
- Topic performance

---

## Inputs

### Student Profile

- Current SAT score
- Target SAT score
- Daily study time
- Test date

### Diagnostic Results

Reading & Writing:
- Information and Ideas
- Craft and Structure
- Expression of Ideas
- Standard English Conventions

Math:
- Algebra
- Advanced Math
- Problem Solving and Data Analysis
- Geometry and Trigonometry

---

## Weakness Identification

For each topic:

accuracy = correct / attempted

Assign:

### Priority 1 (Weak)

Accuracy < 70%

### Priority 2 (Moderate)

70%–85%

### Priority 3 (Strong)

85%+

---

## Question Allocation

Daily questions are distributed based on weakness.

Example:

Weak Topic:
- 20–30 questions

Moderate Topic:
- 10–15 questions

Strong Topic:
- 5–10 questions

---

## Difficulty Progression

Weeks 1–2:
- Easy
- Medium

Weeks 3–5:
- Medium

Final Weeks:
- Medium
- Hard

---

## Practice Tests

Every 2 weeks:

- Full SAT Practice Test

Following Day:

- Test review
- Error logging

---

## Error Log Integration

When a student misses questions:

1. Create error log entry.
2. Categorize mistake.
3. Increase future practice in that topic.

---

## Adaptive Replanning

Whenever new performance data is submitted:

1. Recalculate topic accuracy.
2. Re-rank weaknesses.
3. Regenerate remaining schedule.
4. Preserve completed tasks.

---

## Copyright Compliance

The planner may:

- Recommend topics
- Recommend difficulty
- Recommend question quantities
- Recommend Question Bank filters

The planner may NOT:

- Display SAT questions
- Store SAT questions
- Reproduce SAT passages
- Reproduce answer choices

Students obtain questions directly from College Board.
